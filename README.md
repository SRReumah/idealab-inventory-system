# IDEALab Inventory Management System

A simple web-based application designed to replace the manual component borrowing register at IDEALab.

## Features

- Component request system
- Real-time inventory updates
- Transaction ID generation
- Borrow date and timestamp tracking
- Return workflow with automatic stock restoration
- Status message system
- Modern dark-themed dashboard UI

## How It Works

Students can request components by entering their name and required quantity.  
Each request generates a unique transaction ID and timestamp.

When components are returned, the system updates inventory automatically and closes the transaction.

## Tech Stack

- HTML
- CSS
- JavaScript

This project demonstrates front-end logic for inventory lifecycle management.
